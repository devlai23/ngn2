# Knight Lab Analysis Codebase

This repository contains analysis scripts for the NGN2 dataset in [combined_coded_responses.csv].

The codebase is organized so that each script in [scripts] writes its outputs to a dedicated subfolder under [outputs]. Each script saves readable CSV outputs in addition to any terminal summaries and plots.

## Repository Layout

- [combined_coded_responses.csv]: main survey dataset used by all analyses.
- [scripts]: analysis scripts.
- [outputs]: standardized location for generated CSVs and plots.

## Running Scripts

Every script supports:

```bash
./knight/bin/python scripts/<script>.py --csv combined_coded_responses.csv --outdir outputs/<script>
```

If `--csv` is omitted, the scripts default to `combined_coded_responses.csv`.
If `--outdir` is omitted, the scripts default to `outputs/<script-name>/`.

Examples:

```bash
./knight/bin/python scripts/q1a.py
./knight/bin/python scripts/q1e.py
./knight/bin/python scripts/question_plotting.py
./knight/bin/python scripts/q3_pairwise.py --data combined_coded_responses.csv
```

Note: `q3_pairwise.py` and `q3_index.py` use `--data` rather than `--csv`.

## Script Guide

- [scripts/q1a.py]: Nigeria comfort with AI (`Q6_grid_18`) vs. other countries using Kruskal-Wallis and one-sided Mann-Whitney tests.
- [scripts/q1b.py]: Nigeria fact-checking (`Q6_grid_4`) vs. other countries.
- [scripts/q1c.py]: US and UK lower AI-use tendency (`Q4_grid_10`) vs. other countries.
- [scripts/q1d.py]: US and UK skepticism / human-over-AI preference (`Q6_grid_17`) vs. other countries.
- [scripts/q1e.py]: Nigeria choosing AI tools in `Q12` using overall chi-square and Nigeria-vs-others pairwise tests.
- [scripts/q2a.py]: Q5 platform use by age group with chi-square tests and plots.
- [scripts/q2b.py]: Q8 feed-mix management by age group.
- [scripts/q3b.py]: diverse-language-country grouping test for local plus national news use.
- [scripts/contradiction.py]: overlap and contradiction analysis for Q16 items N and O by country and age.
- [scripts/q3_pairwise.py]: per-item creator-helpfulness and frequency analyses for Q20 and Q21.
- [scripts/q3_index.py]: creator helpfulness index derived from Q20 items.
- [scripts/question_plotting.py]: broad descriptive plots and exported heatmap/bar-chart tables for Q5, Q7, Q8, Q12, Q16, Q6, and Q17.
- [scripts/initial_index_testing.py]: exploratory trust/cynicism summaries and ideology plots.

## Output Convention

Each script writes to its own folder under [outputs]. Typical contents might include:

- `*_summary.csv`: grouped summaries.
- `*_counts.csv`: contingency-table counts.
- `*_rowpct.csv`: row percentages used in plots or statistical interpretation.
- `*_expected.csv`: expected counts from chi-square analyses.
- `*_test.csv` or `*_kruskal_wallis.csv`: omnibus test statistics.
- `*_pairwise*.csv`: pairwise comparison results.
- `*.png`: plots generated by the script.

Example layout:

```text
outputs/
  q1a/
  q1b/
  q1c/
  q1d/
  q1e/
  q2a/
  q2b/
  q3b/
  contradiction/
  q3_pairwise/
  q3_index/
  question_plotting/
  initial_index_testing/
```

## Notes

- The scripts assume survey missing values may appear as `__NA__` or coded values such as `99`.
- Most analyses are country-comparative and use lowercase country codes such as `ng`, `us`, `uk`, `br`, `in`, and `kr`.
- Plot-generating scripts should also export the underlying tables used to create those plots, so results can be reused without re-parsing terminal output.
